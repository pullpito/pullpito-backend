# Pullpito backend

Backend repository for Pullpito.

## Build images
```bash
$ make build-images
```

## Run
```bash
$ make project-up
```

## Stop
```bash
$ make project-down
```

## Database update
```bash
$ make db-update
```

## Run migrations
```bash
$ make db-migration
```

## Run seeds
```bash
$ make db-seed
```

## Test
```bash
$ make run-tests
```

## Logs
```bash
$ make log-monitor
```

# Pullpito backend

Backend repository for Pullpito.

## Run
```bash
$ make project-up
```

## Stop
```bash
$ make project-down
```

## Run migrations
```bash
$ make database-migration
```

## Run seeds
```bash
$ make database-seed
```

## Test
```bash
$ make run-tests
```

## Logs
```bash
$ make log-monitor
```
